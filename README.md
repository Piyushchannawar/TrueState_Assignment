# TrueState Assignment — Full Stack Project

Welcome to the **TrueState Sales Analytics Dashboard** — a full-stack application built using:

- **Backend:** Node.js, Express, MongoDB  
- **Frontend:** React + Vite + TailwindCSS  
- **Deployment:** Render (Backend) & Vercel (Frontend)

This project provides a clean UI to view sales, apply filters, search, paginate, and analyze customer & product data.

## Features (Overview)

- Search sales by customer name or phone  
- Apply multiple filters (Region, Gender, Age, Tags, Category, Payment Method)  
- Reset all filters  
- Statistics cards (Total Units, Total Amount, Total Discount)  
- Pagination  
- Clean UI  
- Ultra-fast Vite-powered frontend  
- Backend deployed on Render  
- Frontend deployed on Vercel
